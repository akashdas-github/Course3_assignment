#Code Book

This code book provides the name of variables present in TidySet.txt data file.

________________________________________________________________________________________________

RepNo -> respondent number
ActivityName -> types of activity when measurments were taken
ActivityNo  -> referance number for activity.
tBodyAcc columns are measurments of the body acceleration signal obtained by subtracting the gravity the gravity from the total acceleration. 
tGravityAcc- columns are measurments of the angular velocity vector measured by the gyroscope for each window sample. The units are radians/second. 
We have data in form of mean and std. dev.
